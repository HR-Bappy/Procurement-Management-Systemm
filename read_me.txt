In this project there is a project planner he/she is able to create a list of school/college/university's
goods need, like for a computer lab of a varsity needs some computers, some monitors, mouse or keyboard etc.
the project planner he/she can make a list of those things with product name, price, quantity, . After that the
 administrator can edit/update the list and can approve that. Another actor called the system admin, he/she has
the ability to add/delete/update the administrator.
